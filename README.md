# professor-check-please

`교수님 확인 부탁드립니다`는 마감 직전의 심정을 패러디한 공감형 밈 서비스이자, 파일 기본 점검을 제공하는 안전한 데모 페이지입니다.

## 서비스 기준

- 실제 제출용 문서 생성 기능을 제공하지 않습니다.
- 손상 파일 생성 기능을 포함하지 않습니다.
- 생성 파일명에는 `[TEST][PARODY]` 접두사가 자동 적용됩니다.
- 업로드 파일은 브라우저에서만 점검하며 서버에 저장하지 않습니다.

## 제공 기능

- **내 파일 바로 점검하기**: 확장자, 용량, 기본 시그니처를 빠르게 확인
- **시연용 샘플 파일 받기**: 설명/시연 전용 TXT, JSON 예시 파일 다운로드

## 실행

```bash
npm install
npm run dev
```

## 기본 검증

```bash
npm run lint
npm run build
```
